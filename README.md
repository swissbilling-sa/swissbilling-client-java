# Swissbilling Java Client

The Java version of the Swissbilling Client library.


## Maven dependency

Add the following to your project `pom.xml`:

```
<dependency>
  <groupId>ch.swissbilling</groupId>
  <artifactId>client</artifactId>
  <version>1.0.0</version>
</dependency>
```
